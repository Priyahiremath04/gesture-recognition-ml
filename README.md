# Multilingual Gesture Recognition System

## Overview
This project is a real-time hand gesture recognition system that uses computer vision and machine learning to detect gestures and convert them into multilingual text and speech output.

## Features
- Real-time gesture detection using MediaPipe  
- Gesture classification using TensorFlow model  
- Multilingual support (English, Kannada, Hindi, Telugu, Tamil)  
- Text-to-Speech output using gTTS  
- Web interface using Flask  

## Technologies Used
- Python  
- OpenCV  
- MediaPipe  
- TensorFlow / Keras  
- Flask  
- NumPy
