# Multilingual Gesture Recognition System

## 🚀 Overview
This project is a real-time hand gesture recognition system that uses computer vision and deep learning to detect gestures and convert them into multilingual text and speech output.

## ✨ Key Features
- Real-time hand tracking using MediaPipe  
- Gesture classification using TensorFlow model  
- Multilingual support (English, Kannada, Hindi, Telugu, Tamil)  
- Text-to-Speech output using gTTS  
- Flask-based web interface  

## 🛠️ Tech Stack
- Python  
- OpenCV  
- MediaPipe  
- TensorFlow / Keras  
- Flask  
- NumPy
